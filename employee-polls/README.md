# Employee Polls

Employee Polls is a React + Redux web application that allows users to create, answer, and view polls.  
Users can log in, vote on questions, create new polls, and view a leaderboard ranking users by participation.
---

## Features

- User authentication (login/logout)
- View answered and unanswered polls
- Vote on polls (one vote per question)
- View poll results with vote counts and percentages
- Create new polls
- Leaderboard ranking users by total activity
- 404 page for invalid routes
- Protected routes (login required)
- Unit tests (including snapshot and DOM tests)

---

## Tech Stack

- React
- Redux
- React Router
- Vite
- Jest
- React Testing Library

---

# Installation & Running the Application

Follow the steps below to run the project locally.
- Commande `npm install`  
- Commande `npm start`  
- Commande `npm test`
